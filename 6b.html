Grammar Reference & Study Notes

Conditional Sentences (Types 1, 2 and 3):

Type Use Structure Example
Type 1 Real possibility in the present/future If + present simple, will + base verb If it rains, we will stay home.
Type 2 Unreal/hypothetical in the present If + past simple, would + base verb If I had money, I would buy a car.
Type 3 Unreal situation in the past If + past perfect, would have + past participle If I had studied, I would have passed.

Wishes (Present & Past):

Type of wish When to use Formula Example
Wish about present Dissatisfaction with current situation S + wish/wishes + S + V2/were I wish I had more time.
Regret about past Regret about something that happened/didn't happen S + wish/wishes + S + had + V3 She wishes she had studied harder.
